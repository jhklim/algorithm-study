# Algorithm Study

단순히 많은 문제를 푸는 것이 아니라  
왜 이 알고리즘을 선택했는지 설명할 수 있는 개발자가 되기 위한 기록입니다.

## Study Goal
- 문제 해결 능력 향상
- 시간복잡도 분석 능력 강화
- 알고리즘 선택 기준 명확화
- 성능 관점에서의 사고 훈련
- 백엔드 개발과 연결되는 구조적 사고 정립

## Study Principles
- 알고리즘 선택 이유 명시 (BFS vs DFS vs Greedy 등)
- 시간복잡도 계산 및 근거 작성
- 개선 포인트 및 리팩토링 기록
- 실패한 접근 및 대안 풀이 비교 정리
- 단순 정답 코드가 아닌 “설계 사고” 중심 정리

## Tech Stack
- Language: Java
- IDE: IntelliJ
- Version Control: Git

## Direction
이 저장소는 단기간 문제 개수를 늘리는 것이 아니라,

문제를 구조적으로 해석하고  
선택의 이유를 설명할 수 있는 개발자로 성장하는 과정을 기록합니다.

단순한 정답이 아닌,  
설계 관점에서 설명 가능한 풀이를 목표로 합니다.
