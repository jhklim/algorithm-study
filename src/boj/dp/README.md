# BOJ 1463 - 1로 만들기

## 문제 요약
정수 N이 주어질 때, 다음 연산을 사용하여 1로 만드는 최소 연산 횟수를 구한다.

- N이 3으로 나누어 떨어지면 3으로 나눈다
- N이 2로 나누어 떨어지면 2로 나눈다
- 1을 뺀다

## 접근 방법
- dp[i]를 i를 1로 만드는 최소 연산 횟수로 정의
- i-1, i/2, i/3에서 오는 경우 중 최소값 선택
- 이전 값들이 이미 계산되어 있어 Bottom-Up DP 방식 사용
- dp[1] = 0 (초기값)

## 점화식
```text
dp[i] = min(
    dp[i-1] + 1,
    dp[i/2] + 1 (i % 2 == 0),
    dp[i/3] + 1 (i % 3 == 0)
)
```
## 시간 복잡도
- O(N)

## 공간 복잡도
- O(N)
