# BOJ 11441 - 구간 합 구하기

## 문제
- N개의 수와 M개의 구간이 주어질 때, 각 구간의 합을 구하는 문제

## 접근 방법
- 누적합(Prefix Sum) 사용

## 시간복잡도
- O(N + M)

# BOJ 11659 - 구간 합 구하기 4

## 접근 방법
1. 크기 N+1의 누적합 배열 S 생성
2. 입력값을 읽으며 누적합 계산
3. 각 쿼리마다 S[e] - S[s-1]을 출력

## 시간 복잡도
- O(N + M)
    - 누적합 계산: O(N)
    - 쿼리 처리: O(M)

# BOJ 11660 - 구간 합 구하기5

N * M 크기의 표가 주어지고,
M개의 질의에 대해서 (x1, y1)부터 (x2, y2)까지 부분 합을 구하는 문제

## 접근 방법
- prefix sum 사용

## 시간 복잡도
O(N^2 + M)
- 누적합 생성: O(N^2)
- 질의 처리: O(M)
