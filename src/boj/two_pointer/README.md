## 접근 방법
- 두 배열은 이미 정렬되어 있으므로 two pointer 사용

## 시간 복잡도
O(N + M)
두 배열을 한번씩 순회

# BOJ 1806 - 부분합

길이가 N인 수욜이 주어질 때,
연속된 부분 수열 중 합이 S이상이 되는 가장 짧은 길이를 구하는 문제
(조건을 만족하는 수열이 없다면 0 출력)

## 접근 방법
- 연속된 부분 수열이므로 슬라이딩 윈도우 사용
- sum >= S -> 길이를 갱신하고 start를 이동해 구간 축소
- sum < S -> end를 이동해 구간 확장
- end가 배열 끝(N)에 도달하면 탐색 종료

## 시간복잡도
O(N)

# BOJ 17609 - 회문

문자열이 주어졌을 때 아래 세 가지 중 하나를 판별
- 0: 회문(Palindrome)
- 1: 유사 회문
- 2: 일반 문자열

## 접근 방법
- two pointer + Recursion 사용

## 핵심 로직
- 처음부터 끝까지 문제 없이 비교 -> 0 (회문)
- 한 번의 문자 제거로 회문 가능 -> 1 (유사 회문)
- 두 번 이상 제거 필요 -> 2 (일반 문자열)

## 시간 복잡도
O(N)

# BOJ 3649 - 로봇 프로젝트

구멍의 너비 x(cm)가 주어지고,
여러 개의 레고 조각 길이 l(nm) 중 두 개를 선택해 합이 정확히 x와 일치하는 조합을 찾는 문제
(가능한 조합이 여러 개라면 두 조각의 길이 차이가 가장 큰 경우를 선택)

## 접근 방법
- 정렬 + two pointer 사용
- 입력은 여러 테스트 케이스로 구성되며, EOF까지 처리

## 시간 복잡도
O(N log N)
