# BOJ 2018 - 수들의 합 5

1부터 N까지 연속된 자연수의 합으로 N을 나타낼 수 있는 경우의 수를 구하는 문제

## 알고리즘 분류
- Two Pointer / Sliding Window

## 시간 복잡도
O(N)

# BOJ 2003 – 수들의 합 2

N개의 양의 정수로 이루어진 수열이 주어질 때,
연속된 부분 수열의 합이 정확히 M이 되는 경우의 수를 구하는 문제

## 알고리즘 분류
- Two Pointer / Sliding Window

## 시간 복잡도
O(N)

# BOJ 3273 - 두 수의 합

정수 배열이 주어졌을 때,
서로 다른 두 수의 합이 특정 값 X가 되는 쌍의 개수를 구하는 문제

## 시간 복잡도
O(N log N)
- 정렬: O(N log N)
- 투 포인터 탐색: O(N)

# BOJ 2559 - 수열

N일 동안 측정된 기온이 주어질 때,
연속된 K일 동안의 기온 합 중 최댓값을 구하는 문제

## 접근 방법
- 윈도우 크기 K값이 항상 고정이기 때문에, while문 에서 조건 분기가 필요 없이 for문으로 모든 위치 탐색

## 시간 복잡도
O(N)

# BOJ 11728 - 배열 합치기

정렬되어 있는 두 개의 배열 A, B 가 주어질 때,
두 배열을 오름차순으로 합쳐 하나의 정렬된 배열로 출력하는 문제

## 접근 방법
- 두 배열은 이미 정렬되어 있으므로 two pointer 사용

## 시간 복잡도
O(N + M)
두 배열을 한번씩 순회

# BOJ 1806 - 부분합

길이가 N인 수욜이 주어질 때,
연속된 부분 수열 중 합이 S이상이 되는 가장 짧은 길이를 구하는 문제
(조건을 만족하는 수열이 없다면 0 출력)

## 접근 방법
- 연속된 부분 수열이므로 슬라이딩 윈도우 사용
- sum >= S -> 길이를 갱신하고 start를 이동해 구간 축소
- sum < S -> end를 이동해 구간 확장
- end가 배열 끝(N)에 도달하면 탐색 종료

## 시간복잡도
O(N)

# BOJ 1940 - 주몽

N개의 재료 번호가 주어졌을 때,
서로 다른 두 재료의 합이 M이 되는 경우의 수를 구하는 문제
(각 재료는 1번만 사용 가능)

## 접근 방법
- 정렬 후 two pointer 사용

## 시간 복잡도
O(N log N)
- 정렬: O(N log N)
- two pointer 탐색: O(N)
