# BOJ 1920 - 수 찾기

N개의 정수가 주어지고,  
M개의 수에 대해 각 수가 배열에 존재하는지 여부를 판단하는 문제.

## 접근 방법
- 입력받은 정수 배열을 오름차순 정렬
- 각 조회 값에 대해 이진 탐색(Binary Search) 수행
- 존재하면 1, 없으면 0 출력

## 선택 이유
- HashSet을 사용하면 평균 O(1)로 해결 가능하지만,
- 이 문제는 정렬 + 이진 탐색이라는 탐색 문제의 기본 패턴을 연습하기 위해
  이진 탐색 방식으로 구현

## 시간 복잡도
- 정렬: O(N log N)
- 탐색: O(M log N)

# BOJ 10815 - 숫자 카드

정수 카드 N개가 주어지고,
M개의 정수에 대해 각 숫자가 카드에 존재하는지 여부를 출력하는 문제
(각 숫자에 대해 존재하면 1, 존재하지 않으면 0)

## 접근 방법
- 주어진 카드 정렬
- 이분 탐색

## 시간 복잡도
O((N+M) log N)
- 카드 정렬 O(N log N)
- 숫자 탐색 O(M log N)
