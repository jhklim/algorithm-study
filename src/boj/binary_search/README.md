# BOJ 1920 - 수 찾기

N개의 정수가 주어지고,  
M개의 수에 대해 각 수가 배열에 존재하는지 여부를 판단하는 문제.

## 접근 방법
- 입력받은 정수 배열을 오름차순 정렬
- 각 조회 값에 대해 이진 탐색(Binary Search) 수행
- 존재하면 1, 없으면 0 출력

## 선택 이유
- HashSet을 사용하면 평균 O(1)로 해결 가능하지만,
- 이 문제는 정렬 + 이진 탐색이라는 탐색 문제의 기본 패턴을 연습하기 위해
  이진 탐색 방식으로 구현

## 시간 복잡도
- 정렬: O(N log N)
- 탐색: O(M log N)

# BOJ 10815 - 숫자 카드

정수 카드 N개가 주어지고,
M개의 정수에 대해 각 숫자가 카드에 존재하는지 여부를 출력하는 문제
(각 숫자에 대해 존재하면 1, 존재하지 않으면 0)

## 접근 방법
- 주어진 카드 정렬
- 이분 탐색

## 시간 복잡도
O((N+M) log N)
- 카드 정렬 O(N log N)
- 숫자 탐색 O(M log N)

# BOJ 10816 - 숫자 카드2

정수 카드 N개가 주어지고,
M개의 정수에 대해 각 숫자가 카드에 몇 개씩 존재하는지 출력하는 문제
(동일한 숫자가 여러 번 등장 할 수 있음)

## 접근 방법
- 주어진 카드 정렬
- 첫 등장 위치(lowerBound), 마지막 등장 다음 위치(upperBound) 계산

## 시간 복잡도
O((N+M) log N)
- 카드 정렬 O(N log N)
- 숫자 탐색 O(M log N)

# BOJ 2805 - 나무 자르기
N개의 나무가 주어질 때,
절단기 높이를 하나 정해서 나무를 자른다.
잘린 나무 길이의 합이 M이상 되도록 하면서,
절단기 높이의 최대값을 구하는 문제

## 접근 방법
- 절단기 높이를 기준으로 이분 탐색 사용

## 시간 복잡도
O(N log H)
- 이분 탐색: O(log H)
- 배열 순회: O(N)

# BOJ 2512 - 예산

여러 지방의 예산 요청 금액과 총 예산 한도 M이 주어질 때,
각 지방에 배정할 상한성을 정하여 모든 지방에 배정된 예산의 합이 M이하 이도록 최대 상한선을 구하는 문제

## 접근 방법
- 상한선 값을 기준으로 이분 탐색 사용

## 시간 복잡도
O(N log M)
- 이분 탐색: O(log M)
- 탐색마다 합 계산: O(N)

# BOJ 1654 - 랜선 자르기

K개의 랜선이 주어졌을 때,
길이가 같은 N개의 랜선을 만들기 위해 자를 수 있는 최대 길이를 구하는 문제

## 접근 방법
- 랜선 길이를 기준으로 이분 탐색 사용

## 시간 복잡도
O(K log L)
- 이분 탐색: O(log L)
- 탐색마다 합 계산: O(K)

# BOJ 2776 - 암기왕

수첩 1에 적혀 있는 수들 중에서
수첩 2에 적힌 각 수가 존재하는지 여부를 확인하는 문제

## 접근 방법
- 수첩 2의 각 숫자에 대해 이분 탐색 

## 시간 복잡도
O((N+M) log N)
- 정렬: O(N log N)
- 각 탐색: O(log N)
