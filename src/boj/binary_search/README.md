# BOJ 1920 - 수 찾기

N개의 정수가 주어지고,  
M개의 수에 대해 각 수가 배열에 존재하는지 여부를 판단하는 문제.

## 접근 방법
- 입력받은 정수 배열을 오름차순 정렬
- 각 조회 값에 대해 이진 탐색(Binary Search) 수행
- 존재하면 1, 없으면 0 출력

## 선택 이유
- HashSet을 사용하면 평균 O(1)로 해결 가능하지만,
- 이 문제는 정렬 + 이진 탐색이라는 탐색 문제의 기본 패턴을 연습하기 위해
  이진 탐색 방식으로 구현

## 시간 복잡도
- 정렬: O(N log N)
- 탐색: O(M log N)

# BOJ 10815 - 숫자 카드

정수 카드 N개가 주어지고,
M개의 정수에 대해 각 숫자가 카드에 존재하는지 여부를 출력하는 문제
(각 숫자에 대해 존재하면 1, 존재하지 않으면 0)

## 접근 방법
- 주어진 카드 정렬
- 이분 탐색

## 시간 복잡도
O((N+M) log N)
- 카드 정렬 O(N log N)
- 숫자 탐색 O(M log N)

# BOJ 10816 - 숫자 카드2

정수 카드 N개가 주어지고,
M개의 정수에 대해 각 숫자가 카드에 몇 개씩 존재하는지 출력하는 문제
(동일한 숫자가 여러 번 등장 할 수 있음)

## 접근 방법
- 주어진 카드 정렬
- 첫 등장 위치(lowerBound), 마지막 등장 다음 위치(upperBound) 계산

## 시간 복잡도
O((N+M) log N)
- 카드 정렬 O(N log N)
- 숫자 탐색 O(M log N)

# BOJ 2805 - 나무 자르기

절단기 높이를 하나 정해서 나무를 자른다.
잘린 나무 길이의 합이 M이상 되도록 하면서,
절단기 높이의 최대값을 구하는 문제

## 접근 방법
- 절단기 높이를 기준으로 이분 탐색 사용

## 시간 복잡도
O(N log H)
- 이분 탐색: O(log H)
- 배열 순회: O(N)

# BOJ 2110 - 공유기 설치

N개의 집 좌표가 주어질 때,
C개의 공유기를 설치하여 인접한 공유기 사이의 최소 거리를 최대화하는 문제

## 접근 방법
- 공유기 사이 거리를 기준으로 이분 탐색
- 가장 왼쪽 집부터 공유기를 설치
- 이전 공유기 위치와의 거리가 D 이상일 경우에만 설치
- 해당 방식으로 C개 이상 설치 가능하면 거리 D는 가능

## 시간 복잡도
O(N log N)
- 정렬: O(N log N)
- 이분 탐색: O(log M)
- 거리 체크: O(N)

# BOJ 2343 - 기타 레슨

N개의 기타 강의를 순서를 유지한 채 M개의 블루레이에 나누어 담을 때,
블루레이의 최소 용량을 구하는 문제

## 접근 방법
- 블루레이 용량을 기준으로 이분 탐색 
- 최소: 가장 긴 강의
- 최대: 모든 강의의 합
- mid를 기준으로 강의들을 담을 때 필요한 블루레이 개수(cnt)를 계산하여 탐색 범위를 조정

## 시간 복잡도
O(N log S)
- 이분 탐색: O(log S) S:강의 길이의 총합
- 탐색마다 합 계산: O(N)

O(M log Max(A))

- 막걸리 양을 기준으로 이분 탐색
